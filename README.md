# astrosage
🚀 AstroSage Analysis Dashboard | Data Analytics Project
Excited to share my latest data analytics project – AstroSage Analysis Dashboard, built to analyze and visualize consultation performance across multiple platforms.
📊 Project Objective:
To analyze call and chat consultation data across different websites (App, Dashboard, Gurucool) and generate actionable business insights.
🔎 Key Insights & Features:
📞 Total Calls Analyzed: 8,365
💬 Total Chat Seconds: 6.8M+
🧾 Total Consultations: 28,027
💰 Revenue Analysis with Net Amount & Astrologer Earnings
⭐ Average Rating: 2.93
📈 Interactive Visualizations Include:
Top 10 Gurus by Rating
Website-wise Consultation Distribution
Hourly Call Volume Distribution
Daily Call Volume Trend
Call & Chat Completion Status
User Distribution by Rating
Dynamic Filters (Website, Consultancy Type, Date)
🛠 Tools Used:
Excel (Advanced formulas, Pivot Tables, Charts, Dashboard Design)
💡 Key Learnings:
Data cleaning & handling missing values
Building KPI metrics
Designing interactive dashboards
Extracting business insights from raw data
This project strengthened my understanding of data visualization, business intelligence concepts, and storytelling with data.
